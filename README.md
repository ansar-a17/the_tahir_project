# The Tahir Project

During Ramadan, I found myself wanting to listen to the right sermon, one that spoke to what I was going through, but faced with over 900 sermons and no way to find it. That problem led to this project.

An open, structured dataset of 927 Friday sermons (Khutbaat-e-Tahir) delivered by Hazrat Mirza Tahir Ahmad, Khalifatul Masih IV (rh), spanning his entire Khilafat from 1982 to 2003.

---

## About the Project

Rather than building one tool, the goal is to enable many. The Tahir Project transforms 927 sermon recordings into a fully structured, searchable dataset. Each sermon has been transcribed, summarised in English, tagged by theme, and annotated with its Quranic citations. The dataset contains over **1.25 million words** of his teachings, now structured, searchable, and freely available for anyone to use, build on, or contribute to.

---

## About Hazrat Mirza Tahir Ahmad

Hazrat Mirza Tahir Ahmad (1928–2003) was the fourth Khalifa of the Ahmadiyya Muslim Community, serving as Khalifatul Masih IV from 1982 until his passing in 2003. Born in Qadian, India, he was educated at Jamia Ahmadiyya and later at the School of Oriental and African Studies (SOAS) in London.

He led the Ahmadiyya Muslim Community through one of its most challenging and transformative periods, including the persecution that forced him into exile in the United Kingdom in 1984. From London, he guided a global community spanning over 200 countries, overseeing the launch of Muslim Television Ahmadiyya (MTA), the world's first 24-hour Islamic television channel, which brought his voice and teachings to millions.

What distinguished his sermons was a rare combination: deep Quranic scholarship, rigorous rationalism, and a willingness to engage directly with science, philosophy, and contemporary world events.

More information is available at [alislam.org](https://www.alislam.org).

## Schema Reference

```
the_tahir_project/
├── README.md                 # Project documentation
├── the_tahir_project.json    # Complete dataset
├── pdfs/                     # PDF files of sermons
├── transcripts/              # Sermon transcriptions
│   ├── [Date].txt            # Full transcription of sermon
│   └── [Date].segments.txt   # Segmented transcription with timestamps
└── youtube_metadata/         # YouTube metadata and information
```

Each sermon entry in `the_tahir_project.json` includes:

| Field | Description |
|---|---|
| `title` | Sermon date (serves as unique identifier) |
| `url` | YouTube video link |
| `duration` | Video length |
| `pdf` | Path to PDF file (empty string if unavailable) |
| `short_summary` | Brief English overview |
| `detailed_summary` | Comprehensive English summary |
| `primary_theme` | Main topic of the sermon |
| `tags` | Relevant keywords and themes (3–8 per sermon) |
| `raw_transcription` | Path to full Urdu transcript |
| `verses` | Quranic verses cited, in chapter:verse format |
| `khilafat_year` | Year of Khilafat at time of delivery |

## How It Was Built

The dataset was built in four stages:

**1. Downloading sermons**
All sermons were downloaded from the Khutbaat-e-Tahir playlist on Youtube.

**2. Audio transcription**
All 927 sermon recordings were transcribed using [Whisper Large v3](https://openai.com/research/whisper), OpenAI's state-of-the-art speech recognition model. An Nvidia A40 GPU was used to transcribe the sermons using Whisper Large V3.

**3. Structured extraction**
Each transcript was passed to Claude Opus 4.8 (Anthropic) via the Batch API for structured extraction. The model generated English short and detailed summaries, identified the primary theme, assigned tags, and extracted Quranic verse citations.

**4. Metadata enrichment**
YouTube metadata, PDF matching, duration, and khilafat year were added programmatically and verified manually where needed.

Total pipeline cost: approximately $100 USD in API usage across both transcription and LLM passes.

## Known Limitations & Quality Notes

**Transcript quality varies.** Audio quality across 20+ years of recordings is inconsistent. Earlier sermons (1982–1987) in particular may have lower audio fidelity. For some reason, several sermons from 2003 are of poor quality. Summaries derived from low-quality transcripts should be treated with caution and are candidates for improvement by contributors.

**Summaries are LLM-generated.** All English summaries were generated by Claude Opus and have not been individually verified by a human scholar. They are accurate to the best of the model's ability but may contain errors in nuanced theological content. Contributions that verify or improve summaries are very welcome.

**The dataset covers khutbaat only.** This project focuses exclusively on Friday sermons. Other addresses, speeches, and writings of Hazrat Mirza Tahir Ahmad are outside the current scope.

## How to Contribute

Contributions are welcome and encouraged. The dataset is only as good as the community that maintains it.

### Adding or Improving Content
- **Summaries**: Help expand or refine sermon summaries and detailed descriptions
- **Tags**: Suggest additional relevant tags or improve theme consistency
- **Transcriptions**: Improve transcript accuracy, particularly Arabic passages
- **Missing sermons**: Help identify and add the 196 PDF-only sermons not yet included

### Reporting Issues
- Report broken links or missing files
- Flag transcription errors or factual inaccuracies in summaries
- Suggest improvements to the schema or data structure

### Process
1. Fork the repository
2. Create a feature branch for your changes
3. Make your improvements following the existing data schema
4. Submit a pull request with a clear description of your changes

Please ensure all changes maintain consistency with the existing data format and structure.
